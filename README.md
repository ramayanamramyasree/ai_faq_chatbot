# AI FAQ Chatbot

## Project Overview

This project is an FAQ chatbot built using Natural Language Processing (NLP).

The chatbot matches user questions with stored AI FAQs and returns the most relevant answer.

## Technologies Used

- Python
- Pandas
- NLTK
- Scikit-Learn

## NLP Techniques

- Tokenization
- Stopword Removal
- Lowercase Conversion
- TF-IDF Vectorization
- Cosine Similarity

## Run Project

python chatbot.py
